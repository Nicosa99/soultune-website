## ARBEITSWEISE

### Grundprinzipien
- **SCHRITT-FÜR-SCHRITT ANSATZ**: Alles was wir angehen, machen wir Schritt für Schritt, dafür vernünftig
- **FOKUSSIERT**: Nicht 50 Ansätze anschneiden, sondern klar der nächste Schritt ausführlich und optimiert
- **STRUKTURIERT**: Bei neuen Sections erstellen wir Code-Sections nach folgendem Ablauf:
  1. HTML erstellen
  2. CSS hinzufügen
  3. JavaScript wenn nötig

### Bricks Code-Editor Workflow
- Wir nutzen die **Code-Editoren im Bricks Builder**
- Jede Section hat **separate Tabs** für:
  - HTML
  - CSS
  - JavaScript
- **Keine vermischten Code-Blöcke** - alles sauber getrennt

### Parallele Entwicklung
1. **Lokale Dateien erstellen** als Enterprise Developer
2. **Parallel in Bricks** mit Code-Sections einpflegen
3. **Versionskontrolle** durch beide Ansätze
